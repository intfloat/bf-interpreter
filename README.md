# bf-interpreter

![Alt text](https://travis-ci.org/intfloat/bf-interpreter.svg?branch=master)

Brainfuck interpreter in Ruby.

## Usage

Simply pass brainfuck source code file as argument, for example:

    ruby ./bf.rb example.bf
